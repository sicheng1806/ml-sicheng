# 新能源电厂调度问题

## 开始之前

### 1. 准备数据集

首先需要从[hf://datasets/sicheng1806/New-energy-power-plant/](https://huggingface.co/datasets/sicheng1806/New-energy-power-plant) 下载数据集至`dataset/csv`文件夹,如下:
```sh
dataset
├── csv
    ├── S1.csv
    ├── W1_power.csv
    └── W1_velocity.csv
```

也可以从`hugging-face`的镜像网站上下载[hf-mirror](https://hf-mirror.com/)。
